T3

Description

给你形如 $\mathrm{Rotate\text{绕点旋转},Trans\text{平移},Scale\text{放缩},Loop\text{循环}}$ 的指令，求一些点经过这些操作后的结果。指令不超过 $1000$ 条，循环允许嵌套。

Solution

矩阵快速幂。

可以发现 $\mathrm{Scale,Trans}$ 两个操作都可以表示成 $[x’ y’ 1]=A[x y 1]$ 的形式，其中 $A$ 为一个矩阵，而稍加计算也可得出 $\mathrm{Rotate}$ 的形式，因此循环就可以用矩阵快速幂解决。注意的是矩阵没有交换率，$\mathrm{Rotate}$ 的矩阵极容易弄错符号！

T4

Description

https://www.luogu.org/problem/P2018 ，其中 $N\le 200000$。

Solution

首先考虑原题怎么做。显然可以枚举传给哪个人，然后贪心一下，先传给总传递时间最大的子树。这样是 $O(n^2\log n)$ 的。

然后再考虑加强版。首先随便选一个人为根，计算出每个人传递给儿子需要的最大时间 $down$。但是如何计算每个人传给父亲的最大时间 $up[i]$？设父亲的 $up$ 值已经知道，那么这个子树的 $up$ 值就是在其他孩子的 $down$ 值与父亲的 $down$ 值的最大值。需要注意的是，设这棵子树在父亲那里按照 $down$ 值排在第 $pos$ 位，那么计算这棵子树的 $up$ 值时，排在他后面的子树的 $down$ 值应减一，因为不需要传给这个儿子了。找排名可以用二分实现，总复杂度 $O(n\log n)$。

吐槽：lower_bound竟比手写二分快。。。TLE自闭了qwq
